## How to sign

Choose **one** of:

1. **Pull request** — Open a PR with your code changes. In the PR description or
   a comment, write: *"I have read the CLA and I sign it."* Include your full
   legal name and the date.

2. **Dedicated CLA PR** — Open a PR titled `CLA: Your Full Name` that adds your
   name to a `CONTRIBUTORS.md` file (if the project maintains one) or only
   contains your signed-off commit message (see below).

3. **Signed-off commit** — Use Git's sign-off line on each commit:

   ```bash
   git commit -s -m "Your change description"
   ```

   The sign-off certifies the Individual CLA terms below.
