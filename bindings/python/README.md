# Python binding — nativeview

Package: **`nativeview/`** (ctypes)  
Metadata: **`pyproject.toml`**

**Full guide:** [docs/Python.md](../../docs/Python.md)

**Examples:** [examples/python/](../../examples/python/), [examples/todo_app/py_todo/](../../examples/todo_app/py_todo/)

Requires shared `libnativeview` (`-DNV_BUILD_SHARED=ON`) or set **`NATIVEVIEW_LIB`** to the library path.

```bash
pip install -e bindings/python   # from repo root
# or
export PYTHONPATH=/path/to/nativeView/bindings/python
```
